# UYAP Robotu
UYAP sisteminde tarayıcı eklentisi olarak çalışan ve tüm ihaleleri size kazandıran gelmiş geçmiş en iyi bot yazılımı.

## Özellikler
- Limit belirleyin
- Artırım miktarını belirleyin
- Mikrosaniyelerde işlemlerle %100 ihaleyi alma garantisi.
